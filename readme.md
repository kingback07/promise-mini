# 一言为定 (Promise Mini) - 使用说明书

你好！欢迎使用“一言为定”。这是一个帮你记录承诺的小工具。
就像我们平时说的“拉钩上吊一百年不许变”，这里可以把你的每一次“拉钩”都记下来。

## 这个小程序能做什么？

1.  **记下承诺**：
    比如“我要每天背10个单词”或者“这周末要帮妈妈做家务”。你可以把它写下来，设个时间。
2.  **查看承诺**：
    打开小程序，你就能看到自己许下了哪些诺言，哪些还在进行中，哪些已经完成了。
3.  **完成打卡**：
    当你做到了，就可以回来点个“完成”，给自己一个小小的鼓励！

## 怎么开始使用？

如果你是开发者或者想学习怎么做这个小程序，请看下面：

### 第一步：准备工作
1.  你需要电脑上安装了“微信开发者工具”。
2.  把这个项目文件夹导入到工具里。

### 第二步：开启云服务
这个小程序用到了“云开发”，就像是有个云端的日记本帮你存数据。
1.  在开发者工具里点击“云开发”按钮，开通一下（选免费的就行）。
2.  找到你的环境ID（一串字母和数字）。
3.  打开 `miniprogram/envList.js` 文件，把原来的ID换成你自己的。

### 第三步：上传云函数
云函数就像是云端的小帮手。
1.  找到 `cloudfunctions` 文件夹。
2.  右键点击里面的文件夹（比如 `login`），选择“上传并部署：云端安装依赖”。
3.  把所有的云函数都上传一遍。

### 第四步：开始运行
点击工具里的“编译”，你就可以在模拟器里看到小程序啦！

## 功能列表 (给产品经理看)

| 功能名称 | 描述 |
| :--- | :--- |
| **我的承诺** | 首页展示所有承诺列表，区分“进行中”和“已完成”。 |
| **新建承诺** | 点击“+”号，填写标题、内容和截止时间，创建一个新承诺。 |
| **完成承诺** | 在列表里，可以把承诺标记为“已完成”。 |
