# 一言为定 (Promise Mini) - 小程序项目文档

## 1. 项目简介
“一言为定”是一款基于微信小程序生态的工具类应用，旨在帮助用户记录承诺、契约或约定。通过数字化记录，增强人与人之间的信任与契约精神。

**核心价值**：
- **记录**：便捷记录承诺内容、时间、对象。
- **见证**：虽然目前主要由用户自行记录，未来可引入见证机制。
- **提醒**：承诺到期提醒。

## 2. 技术架构

本项目采用 **微信小程序云开发 (WeChat CloudBase)** 方案，实现免服务器运维、低成本快速上线。

### 2.1 架构概览
- **前端**：微信小程序原生框架 (WXML, WXSS, JS/TS)
- **后端**：微信云开发 (Cloud Functions)
- **数据库**：云数据库 (Cloud Database - NoSQL)
- **存储**：云存储 (Cloud Storage) - 用于存储图片/文件证据

### 2.2 目录结构
```
/workspace
  ├── cloudfunctions/        # 云函数目录
  ├── miniprogram/           # 小程序前端代码
  │   ├── components/        # 公共组件
  │   ├── images/            # 图片资源
  │   ├── pages/             # 页面文件
  │   ├── style/             # 全局样式
  │   ├── utils/             # 工具函数
  │   ├── app.js             # 小程序逻辑入口
  │   ├── app.json           # 小程序全局配置
  │   ├── app.wxss           # 小程序全局样式
  └── project.config.json    # 项目配置文件
```

## 3. 功能规划 (V1.0)
1.  **承诺列表**：展示我发出的和收到的承诺。
2.  **创建承诺**：填写承诺内容、截止时间、相关人。
3.  **承诺详情**：查看承诺详细信息及状态（进行中、已达成、未达成）。

## 4. 快速开始
1.  使用微信开发者工具导入本项目根目录。
2.  开通云开发环境，并在 `miniprogram/envList.js` 中配置环境 ID。
3.  上传并部署 `cloudfunctions` 中的云函数。
4.  编译运行。

## 5. 开发规范
- 遵循 `Architecture.md` 中的架构设计原则。
- 代码注释需清晰，变量命名语义化。
- 提交代码前确保无明显报错。
